
Rhience Portfolio - GitHub Pages Instructions
---------------------------------------------
1. Delete all old files from your GitHub repo.
2. Upload all files from this ZIP (do NOT upload the folder itself).
3. Commit changes.
4. Enable GitHub Pages if not already active.
5. Wait 1-5 minutes, then visit https://rhience.github.io.

To update:
- Edit index.html for text content.
- Replace images/logos in assets/ folder.
- Replace Rhience-Asuncion-Resume.pdf for updated resume.
- Edit style.css for design changes.
- Edit script.js for animations or effects.

Enjoy your portfolio!
